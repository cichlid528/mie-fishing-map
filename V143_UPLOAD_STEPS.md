# v143・釣りニャン吹き出し文字内側調整版 アップロード手順

## 変更内容

- 吹き出し画像は v142 の右下しっぽ版を継続
- 吹き出しを正方形比率で表示し、画像の潰れを防止
- 文字エリアを吹き出しの内側に収めるよう余白を再調整

- 宮川ダム、七色ダム、池原ダムの修正は継続

## アップロード方法

1. このZIPを解凍します。
2. 中身を GitHub リポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. `assets` フォルダも必ず一緒にアップロードします。
4. コミット名は下記がおすすめです。

```text
釣りニャンの吹き出し文字を内側に収める
```
