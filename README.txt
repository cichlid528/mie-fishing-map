三重県バス釣りマップ

開き方:
1. 「開く.bat」をダブルクリックします。
2. または index.html をブラウザで開きます。

地図について:
- 国土地理院の「地理院タイル」を使用しています。
- APIキーや課金設定は不要です。
- 地図右上で「標準地図」と「航空写真」を切り替えられます。
- 池の名称と位置は、国土地理院の名称検索と三重県ため池データベースを照合しています。
- 出典: 国土地理院

内容:
- 左側に三重県内の川・池・ダムの初期リストがあります。
- 池・湖・貯水池・公園池・お堀系の候補を初期リストに入れています。
- 各場所ごとに「ブラックバスがいる」「ブラックバスがいない」「釣り禁止」「駐車スペース」をチェックできます。
- 「釣り場追加」を押して地図をタップすると、名前がない池や水辺を自分で釣り場として追加できます。
- 自分で追加した釣り場には、名称、種別、地名、メモを保存できます。
- 自分で追加した釣り場は、選択後のカードまたは編集画面から削除できます。
- 「釣果ピン追加」を押して地図をタップすると、赤い釣果ピンを追加できます。
- 釣果ピンには日時、ルアー、天気、風、水の状態、サイズ、メモを保存できます。
- 釣果ピンには写真を添付できます。写真は圧縮してこの端末内に保存されます。
- 左側の「釣果」タブで釣果ピンの一覧を確認できます。
- 釣果ピンには保存順の通し番号が表示されます。
- チェック内容は同じブラウザ内に保存されます。
- 自分で追加した釣り場も同じブラウザ内に保存されます。
- 釣果ピンも同じブラウザ内に保存されます。
- 左の名称を押すと、右側の地図がその場所へ移動します。

PWAとして使う方法:
1. index.html を直接ファイルで開くのではなく、ローカルサーバーまたはHTTPS環境で開きます。
2. iPhoneではSafariでアプリURLを開きます。
3. 共有ボタンを押します。
4. 「ホーム画面に追加」を選びます。
5. ホーム画面の「釣り場ログ」アイコンから起動します。

ローカル確認の例:
- このフォルダをローカルサーバーで配信して、SafariやChromeで開きます。
- 例: http://localhost:8000/ のようなURLで開くと、service workerを確認しやすくなります。
- file:/// で直接開いた場合、PWAやservice workerは基本的に動きません。

PWA追加ファイル:
- manifest.json
- sw.js
- icons/icon-192.png
- icons/icon-512.png
- icons/icon-maskable-512.png
- icons/apple-touch-icon.png

アイコンについて:
- 現在の icons フォルダ内の画像は仮アイコンです。
- 好きなデザインにする場合は、同じファイル名・同じサイズのPNGで差し替えてください。

注意:
- 初期リストの位置は地図上で扱いやすい代表地点です。
- 釣り禁止や立入禁止は変わることがあるため、現地の看板、管理者、自治体の情報を必ず確認してください。
- 地図表示にはインターネット接続が必要です。
- 名前が登録されていない小さな池や水辺は、今後「自分で釣り場を追加する」機能で登録できるようにする想定です。
- 位置情報など一部のスマートフォン機能はHTTPS環境が必要になる場合があります。
- 現在の保存方式はlocalStorageです。Safariのデータ削除、端末変更、ブラウザ変更でデータが消える可能性があります。
- 将来的にはJSONエクスポート/インポート機能、クラウド同期、バックアップ機能を追加するのがおすすめです。
