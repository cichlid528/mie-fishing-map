# v147・釣りニャン釣果記録ボタン連動版 アップロード手順

今回の変更:

- 釣りニャンの吹き出し内「釣果記録」ボタンから、アプリ本体の釣果記録画面を直接開くように変更しました。
- 開く位置は現在の地図中央です。
- 画面内の「現在地を使う」ボタンを押せば、現在地で記録できます。
- これまでの釣りニャン反応モード、吹き出し調整、宮川ダム・七色ダム・池原ダム修正も継続しています。

## アップロード方法

1. このZIPを解凍します。
2. 中身を GitHub リポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. `assets` フォルダも必ず一緒にアップロードしてください。
4. コミット名は以下でOKです。

```text
釣りニャンの釣果記録ボタンを本体記録画面に連動
```

## アップロード後の確認

まずキャッシュ削除:

```text
https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1
```

確認URL:

```text
https://cichlid528.github.io/mie-fishing-map/?v=v147-pet-catch-record-button
```

確認ポイント:

- 釣りニャンの吹き出しを開く
- 「釣果記録」ボタンを押す
- 地図をタップ待ちではなく、釣果記録フォームが直接開く
- 必要ならフォーム内の「現在地を使う」で現在地に変更できる
