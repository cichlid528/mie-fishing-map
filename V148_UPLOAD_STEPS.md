# v148・釣りニャン釣果記録ボタン高速化版 アップロード手順

今回の変更:

- 釣りニャンの吹き出し内「釣果記録」ボタンを押した時の待ち時間を短くしました。
- 以前の fallback で現在地取得ボタンを押してしまい、GPS待ちで次画面まで長くなることがあったため、その処理をやめました。
- アプリ本体の釣果記録画面を直接開く橋渡し処理が準備できるまで、短時間だけ待ってから開きます。
- 橋渡しがまだ使えない場合も、GPS待ちには入らず「地図をタップして記録」モードへ切り替えます。
- v146/v147までの釣りニャン演出、宮川ダム・七色ダム・池原ダム修正も継続しています。

## アップロード方法

1. このZIPを解凍します。
2. 中身を GitHub リポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. `assets` フォルダも必ず一緒にアップロードしてください。
4. コミット名は以下でOKです。

```text
釣りニャンの釣果記録ボタンの待ち時間を短縮
```

## アップロード後の確認

まずキャッシュ削除:

```text
https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1
```

確認URL:

```text
https://cichlid528.github.io/mie-fishing-map/?v=v148-pet-catch-button-fast
```

確認ポイント:

- 釣りニャンの吹き出しを開く
- 「釣果記録」ボタンを押す
- GPS待ちで長く止まらず、すぐ釣果記録画面が開く
- フォーム内の「現在地を使う」を押した時だけ現在地取得に入る
