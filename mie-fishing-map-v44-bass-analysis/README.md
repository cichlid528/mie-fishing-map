# 三重県釣りマップ v44 野池バス分析版

GitHub Pagesでそのまま公開できる、ビルド不要の静的アプリです。

## 追加した内容

v43の流れを壊さないよう、保存キーを分けたまま、釣果記録に野池バス向けの分析項目を追加しています。

- 水位: 満水、普通、減水、大減水、増水
- ベイト確認: ギル、小魚、エビ、カエル、虫、不明
- カバー・地形: アシ、オーバーハング、沈み木、護岸、ウィード、ブレイク、流れ込み、堰堤、シェード
- 反応場所: 岸際、ブレイク、流れ込み、堰堤、シェード、カバー奥、沖
- プレッシャー: 低い、普通、高い、人多い、先行者あり、足跡あり
- 時間帯: 朝、昼、夕方、夜
- ルアー名、色、重さ
- サイズ実寸cm
- 40up集計

## 保存キー

公開中v43で使っている保存キーに合わせています。

```text
mie-bass-map-v1
mie-bass-catches-v1
mie-bass-custom-spots-v1
mie-fishing-map-sidebar-background-v1
mie-fishing-map-position-overrides-v7
```

以前のGitHub編集用ZIPで使った `mieFishingMap.v1` が端末にある場合は、初回起動時に釣り場と記録を新しい保存キーへ移します。

## 公開方法

1. ZIPを展開します。
2. 中身をGitHubリポジトリへアップロードします。
3. GitHubの `Settings` → `Pages` で公開ブランチを選びます。
4. 公開URLを開きます。

Service Workerのキャッシュ名は `bass-spot-log-v44` です。更新後に古い画面が残る場合は、ブラウザで再読み込みするか、PWAを一度閉じて開き直してください。

## 注意

掲載情報は釣りの許可を意味しません。現地看板、管理者、自治体、漁協などの最新情報を必ず確認してください。
