# 三重県釣りマップのダウンロード・インストール方法

三重県釣りマップは、GitHub Pagesで公開しているPWAアプリです。
スマホやPCにインストールして、アプリのように使えます。

## アプリを開く

最新版確認URL：

https://cichlid528.github.io/mie-fishing-map/?v=v168-record-button-fix

キャッシュ削除後に開く場合：

https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1

## スマホにインストールする方法

### Android / Chrome

1. 上のアプリURLをChromeで開く
2. 右上の「︙」メニューを押す
3. 「ホーム画面に追加」または「アプリをインストール」を押す
4. 追加を押す

### iPhone / Safari

1. 上のアプリURLをSafariで開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. 追加を押す

## GitHubからソースをダウンロードする方法

GitHubのリポジトリ画面で、次の操作をしてください。

1. 緑色の「Code」ボタンを押す
2. 「Download ZIP」を押す
3. ZIPを保存する

## 注意

- 釣果記録や追加したポイントは、基本的に使用している端末のブラウザ内に保存されます。
- 機種変更やブラウザのデータ削除前には、アプリ内の「バックアップ保存」を使ってください。
- 表示がおかしい場合は、先に `reset-cache.html?auto=1` を開いてキャッシュを削除してください。
