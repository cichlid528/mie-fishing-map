# 三重県釣りマップ

三重県の釣り場、釣果、写真、メモを自分専用の記録として残せる**無料**の地図アプリです。会員登録不要、広告・アプリ内課金なし。

## 公開中のアプリ

- [三重県釣りマップを開く](https://cichlid528.github.io/mie-fishing-map/)
- [インストール方法を見る](https://cichlid528.github.io/mie-fishing-map/install.html)
- [ソースコードをZIPでダウンロードする](https://github.com/cichlid528/mie-fishing-map/archive/refs/heads/main.zip)
- [不具合を報告する](https://github.com/cichlid528/mie-fishing-map/issues)

> GitHubのDownload ZIPは**ソースコード**です。スマホへの追加はインストール方法をご利用ください。APK・IPAは配布していません。

## Android
Chromeでアプリを開き、インストール表示またはメニューの「アプリをインストール」「ホーム画面に追加」を選びます。

## iPhone・iPad
Safariで開き、共有ボタン（四角から上向き矢印）から「ホーム画面に追加」→「追加」を選びます。アプリ内ブラウザではなくSafariを使ってください。

## パソコン
Windows、Mac、Chromebookではブラウザからそのまま無料で利用できます。対応ブラウザではブラウザのインストール機能も使えます。

## 保存データと注意
釣果・写真・メモ・追加した釣り場は基本的に利用端末内に保存されます。ブラウザの保存データを消すと記録が消える可能性があり、機種変更時の自動移行は保証されません。釣り禁止・立入禁止区域、現地の看板、管理者、漁協の指示を優先し、安全と私有地・駐車場所に注意してください。
