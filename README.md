# v172・釣り場チェック欄復旧版

この修正版は、釣り場一覧から消えていた以下のチェック欄を復旧します。

- 釣れた
- 魚種
- 禁止
- 駐車

## 修正内容

- 釣り場リストを5列表示に戻しました。
- 釣り場ごとの「釣れた」「禁止」「駐車」チェックを保存できるようにしました。
- 「魚種」ボタンから魚種を入力・保存できるようにしました。
- 保存先は `mie-bass-map-v1` です。
- 既存の釣り場ポイント、追加ポイント、釣果記録、写真データは削除しません。
- 釣果記録を保存したとき、近くの釣り場が選ばれている場合は「釣れた」と魚種も連動保存します。
