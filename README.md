# 三重県釣りマップ

三重県の釣り場と釣果を記録するための地図アプリです。  
野池バス釣り向けに、水位・ベイト・カバー・反応場所・プレッシャー・ルアー詳細・実寸サイズなどを記録できます。

## アプリを開く

下のリンクからアプリを開けます。

[三重県釣りマップを開く](https://cichlid528.github.io/mie-fishing-map/)

## アプリとしてダウンロード・追加する方法

このアプリは、App Store や Google Play から入れるアプリではなく、ブラウザからホーム画面に追加して使う PWA です。

### Android / Chrome

1. Android の Chrome で [三重県釣りマップ](https://cichlid528.github.io/mie-fishing-map/) を開く
2. 左メニューの「アプリをダウンロード」を押す
3. インストール画面が出たら「インストール」を押す
4. 出ない場合は、Chrome 右上の「︙」から「アプリをインストール」または「ホーム画面に追加」を選ぶ

### iPhone / Safari

1. iPhone の Safari で [三重県釣りマップ](https://cichlid528.github.io/mie-fishing-map/) を開く
2. 画面下の共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. 右上の「追加」を押す

※ iPhone は仕様上、Android のような自動インストール画面は出にくいです。Safari の「ホーム画面に追加」を使ってください。

## 記録できる主な内容

- 釣り場追加
- 現地メモ登録
- 記録ピン追加
- 釣果写真の添付
- 天気、風、水の状態
- 水位
- ベイト確認
- カバー・地形
- 反応場所
- プレッシャー
- 時間帯
- ルアー名、色、重さ
- サイズ実寸 cm
- 40up 集計
- バックアップ保存 / 読込

## 大事な注意

掲載されている釣り場情報は、釣りの許可を意味するものではありません。  
釣行前に、現地看板・管理者・自治体・漁協などの最新情報を必ず確認してください。

私有地への無断立ち入り、迷惑駐車、ゴミの放置は禁止です。  
禁止・危険・立入制限の表示がある場所では釣りをしないでください。

## データ保存について

釣果、写真、追加した釣り場、背景画像などは、基本的にこの端末内に保存されます。  
端末変更、ブラウザのデータ削除、Safari / Chrome の保存領域整理で記録が消えることがあります。

大事な記録は、アプリ内の「バックアップ保存」から JSON ファイルとして保管してください。

## 更新が反映されないとき

アプリを更新したのに古い表示が残る場合は、次を試してください。

1. ブラウザで再読み込みする
2. URL の末尾に `?v=45` などを付けて開く
3. ホーム画面から開いている場合は、一度ブラウザで開き直す
4. それでも直らない場合は、ブラウザのサイトデータまたはキャッシュを削除する
