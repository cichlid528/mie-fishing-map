# 三重県釣りマップ v171-add-spot-button-fix

釣り場追加ボタンを復旧する版です。

## 修正内容

- 「釣り場追加」ボタンを押すと釣り場追加パネルが開くように修正
- 地図中央の座標で釣り場を追加できるように修正
- 釣り場名・種別・地名・メモを保存できるように修正
- 保存後、マップピンと釣り場一覧に即反映
- 追加釣り場を `mie-bass-custom-spots-v1` に保存
- 既存の釣り場、釣果記録、写真データは削除しない

## バージョン

`v171-add-spot-button-fix`

表示ラベル：`v171・釣り場追加ボタン修正版`
