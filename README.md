# v169 写真アップロード・カメラ機能復旧版

釣果記録の写真フォルダー選択、カメラ撮影、写真プレビュー、写真付き保存を復旧しました。

## バージョン

`v169-photo-camera-fix`

## 主な修正

- `catchPhoto` 写真フォルダー選択のイベントを復旧
- `catchCamera` カメラ撮影のイベントを復旧
- 写真をJPEGに圧縮して端末のlocalStorageへ保存
- 写真プレビュー表示を復旧
- 「写真を外す」ボタンを復旧
- 保存した写真を記録一覧と釣果ピンのポップアップに表示
- 背景画像変更の写真フォルダー選択・カメラ撮影も復旧
- 既存の釣果記録、釣果写真、釣り場ポイントは削除しない

## 重要

`index.html` の読み込みを必ず `v169-photo-camera-fix` に変更してください。
