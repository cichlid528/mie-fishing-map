## 重要
- 地図は国土地理院の標準地図タイルを使います。
- 初期ポイント `seedSpots` をアプリ内に直接持たせ、外部GitHub fetchに依存しないようにしています。
- `localStorage` の釣り場・釣果・写真・メモは削除しません。
- `app.js` と `app-v156-loader-fixed.js` は同じ復旧用アプリ本体です。どちらを index.html が読んでも動くようにしています。
