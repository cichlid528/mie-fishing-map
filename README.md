# 三重県釣りMAP

三重県の池、川、ダム、漁港、港、マリーナを確認し、釣果や現地情報を端末に記録できる地図アプリです。

## 公開中のアプリ

[三重県釣りMAPを開く](https://cichlid528.github.io/mie-fishing-map/)

## スマートフォンにアプリとして追加

### Android（Chrome）

1. Chromeで[三重県釣りMAP](https://cichlid528.github.io/mie-fishing-map/)を開きます。
2. 画面下の「インストール」を押します。
3. 確認画面で「インストール」または「ホーム画面に追加」を選びます。

Chromeのメニューから「アプリをインストール」または「ホーム画面に追加」を選ぶこともできます。

### iPhone（Safari）

1. Safariで[三重県釣りMAP](https://cichlid528.github.io/mie-fishing-map/)を開きます。
2. 画面下の共有ボタンを押します。
3. 「ホーム画面に追加」を選び、右上の「追加」を押します。

### GPS

初回利用時に位置情報の使用を許可すると、地図上に現在地を表示できます。

## 背景画像を変更

1. 左メニューの「背景画像を変更」を押します。
2. 「写真を撮影して背景にする」または「写真フォルダーから選ぶ」を選びます。
3. 選んだ画像は自動で縮小・圧縮され、この端末に保存されます。

「初期背景に戻す」を押すと、最初の背景画像へ戻ります。
