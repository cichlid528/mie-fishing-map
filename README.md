# v167・釣果記録と釣果ピン復旧版

## 目的
国土地理院マップと釣り場ピンに加えて、消えたように見える釣果記録・釣果ピンを復旧します。

## 復旧内容
- `mie-bass-catches-v1` の釣果記録を読み込みます。
- 旧形式 `mieFishingMap.v1` 内の `records` / `catches` も読み込みます。
- 位置情報のある釣果はオレンジ色の釣果ピンとして表示します。
- 位置情報がない釣果も、記録一覧には残して表示します。
- `spotId` が残っている記録は、釣り場ポイントの座標から釣果ピン位置を復旧します。
- 釣果、写真、メモ、追加ポイントは削除しません。

## 注意
`index.html` の読み込み先を必ず `v167-catch-records-restore` に変更してください。
