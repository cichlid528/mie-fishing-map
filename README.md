# v176・釣り場魚種選択式復旧版

「釣り場一覧 / 釣り場カード」の魚種ボタンを、文字入力のpromptではなく、選択式パネルに変更しました。

## 修正内容

- メニューの釣り場一覧にある「魚種」ボタンを押すと、魚種選択パネルが開きます。
- 淡水 / 海水 / イカ・タコ・エビ / すべて に分けて選択できます。
- 複数魚種を選択できます。
- 保存先は既存の `mie-bass-map-v1` です。
- 釣れた / 禁止 / 駐車 / 釣果記録 / 写真 / 爆釣にゃん師匠モーションは削除しません。
