# 三重県釣りMAP

三重県の池、川、ダム、漁港、港、マリーナを確認し、釣果や現地情報を端末に記録できる地図アプリです。

## 公開URL

https://cichlid528.github.io/mie-fishing-map/

## スマートフォンへの追加

AndroidではChromeに表示される「インストール」ボタン、iPhoneではSafariの共有メニューにある「ホーム画面に追加」からアプリとして追加できます。

GPSはHTTPSで公開したページを開き、端末の位置情報を許可すると利用できます。
