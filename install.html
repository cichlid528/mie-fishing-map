三重県釣りマップ

無料・会員登録不要

三重県の釣り場、釣果、写真、メモを自分だけの記録として残せる地図アプリです。

三重県釣りマップを開く

ブラウザからそのまま無料で利用できます。

Androidへの追加方法

  1. Chromeでアプリを開く
  2. 画面右上のメニューを押す
  3. 「アプリをインストール」または「ホーム画面に追加」を選ぶ
  4. 確認画面で「インストール」を押す

端末やChromeの版により表示名が異なる場合があります。

iPhone・iPadへの追加方法

  1. Safariでアプリを開く
  2. 共有ボタンを押す
  3. メニューを下へ移動する
  4. 「ホーム画面に追加」を押す
  5. 右上の「追加」を押す

共有ボタンは四角から上向き矢印のマークです。ChromeやLINEなどのアプリ内ブラウザではなくSafariで開き直してください。

パソコンでの利用

Windows、Mac、Chromebookではブラウザからそのまま利用できます。対応ブラウザではアドレスバーやメニューからインストールも可能です。

パソコンで三重県釣りマップを開く

注意事項

釣り禁止・立入禁止区域には入らず、現地の看板、管理者、漁協の指示を優先してください。天候、足場、駐車場所、私有地に注意し、安全を優先してください。記録は基本的に端末内へ保存され、ブラウザ保存データを削除すると消える場合があります。広告・課金・会員登録はありません。
