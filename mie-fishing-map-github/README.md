# 三重県釣りマップ - 野池メモ版

GitHub Pages でそのまま公開できる、ビルド不要の静的Webアプリです。
釣り場、野池、釣果、現地メモ、周辺メモを端末内に保存できます。

## 入っている機能

- 地理院地図ベースの地図表示
- 釣り場追加・編集・削除
- 種別フィルター：池 / 川 / ダム / 港・漁港 / マリーナ
- 検索：名称、メモ、魚種、ルアー、条件など
- 記録ピン追加
- 現在地メモ登録
- 釣果記録
  - 魚種
  - サイズ区分、実寸cm
  - ルアー系統、ルアー名、色・重さ
  - 天気、風、水の状態、水位
  - ベイト確認：ギル、小魚、エビ、カエル、虫など
  - カバー・地形：アシ、オーバーハング、沈み木、護岸、ウィードなど
  - 反応場所：岸際、ブレイク、流れ込み、堰堤、シェードなど
  - プレッシャー：人多い、先行者、足跡、静かなど
  - 時間帯：朝、昼、夕方、夜
  - 写真添付と圧縮保存
- 40upブラックバス件数の集計
- バックアップJSON保存・読込
- PWA対応：スマホのホーム画面に追加しやすい構成

## GitHubで編集して公開する方法

### 1. 新しいリポジトリを作る

GitHubで新規リポジトリを作ります。
例：`mie-fishing-map`

### 2. ZIPを展開してアップロード

このZIPを展開して、中身のファイルをリポジトリ直下にアップロードします。

重要：`index.html` がリポジトリの一番上に来るようにしてください。

```text
mie-fishing-map/
├─ index.html
├─ style.css
├─ app.js
├─ manifest.webmanifest
├─ sw.js
├─ assets/
└─ data/
```

### 3. GitHub PagesをONにする

GitHubのリポジトリ画面で、

`Settings` → `Pages` → `Build and deployment` → `Deploy from a branch`

を選びます。

- Branch: `main`
- Folder: `/root`

で保存します。

数分後に公開URLが表示されます。

## 編集する場所

### 画面の文字を変えたい

`index.html` を編集します。

### 色や見た目を変えたい

`style.css` を編集します。

特に以下を変えると雰囲気が変わります。

```css
:root {
  --accent: #0f766e;
  --accent-dark: #0b5b55;
}
```

### 記録項目を増やしたい

`index.html` のフォーム項目を追加し、`app.js` の `saveRecordForm()` と `openRecordDialog()` に項目を追加します。

### 初期釣り場データを入れたい

今の版は、プライベートな野池情報を守るため、初期データは空です。
自分の端末で釣り場を追加してから、バックアップ保存でJSONを書き出してください。

## 注意

このアプリは、釣り場の許可情報を保証しません。
釣行前に現地看板、管理者、自治体、漁協の最新情報を必ず確認してください。

私有地への無断立ち入り、迷惑駐車、ゴミの放置は禁止です。

## データ保存について

データはブラウザの `localStorage` に保存されます。
スマホのブラウザデータ削除、機種変更、ストレージ整理で消える場合があります。
大事な記録は、必ず「バックアップ保存」でJSONを保管してください。
