# v137・釣りニャン表情モーション採用版 アップロード手順

## 変更内容

- 今回もらった「爆釣にゃん師匠」モーション提案画像を採用
- 立ち姿・横向き・後ろ姿・しゃがみ姿を追加
- 表情バリエーション12種類を追加
  - 笑顔
  - キリッ
  - ウインク
  - 驚き
  - 考え中
  - やったー
  - 残念
  - 怒り
  - ドヤ顔
  - うっとり
  - 焦り
  - ねむい
- 吹き出しに「表情」ボタンを追加
- タップ時やメッセージ表示時に、表情がランダムで切り替わるように変更
- 後ろ姿の「見回り」モーションも継続
- 大杉湖 → 宮川ダム
- 七色貯水池 → 七色ダム
- 池原ダムを追加
- バージョンを `v137-turi-nyan-motion-sheet` に更新

## 必ずアップロードするファイル

ZIPを解凍して、中身を GitHub リポジトリ `mie-fishing-map` の直下へ上書きアップロードしてください。

- `app.js`
- `pwa-install.js`
- `manifest.json`
- `reset-cache.html`
- `sw.js`
- `assets/` フォルダ一式

`assets` フォルダの中に、釣りニャンの表情画像とモーションシートが入っています。

## コミット名

```text
釣りニャンの表情モーションを採用
```

## 確認ポイント

- 右下に「爆釣にゃん師匠」が表示される
- 吹き出しに「表情」ボタンがある
- 「表情」を押すと、笑顔・ウインク・考え中などに変わる
- 「見回り」を押すと後ろ姿になる
- 七色貯水池が七色ダムになっている
- 池原ダムが表示されている
