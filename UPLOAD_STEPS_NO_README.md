## v123 変更内容

- 「地図ラベル名で池候補追加」ボタンに変更しました。
- 地図上の「○○池」ラベルを画面中央に合わせてボタンを押すと、その中央位置へ池候補ポイントを追加します。
- 国土地理院の池名ラベルを取得できた場合は、その池名で池候補を追加します。
- 池名ラベルを取得できない場合は、池名入力画面を出して、入力した名前で画面中央へ追加できます。
- 追加後は池候補表示に切り替え、追加したポイントへズームして詳細カードを開きます。

## 確認方法

1. マップを開く。
2. 追加したい「○○池」の文字が画面中央付近に来るまで地図を動かす。
3. 「地図ラベル名で池候補追加」を押す。
4. 池候補ポイントが追加され、詳細カードが開くことを確認する。
5. 名前が自動取得できなかった場合は、表示された入力欄に「一ノ口池」のように池名を入力してください。
