# v121 アップロード手順

1. ZIPを解凍します。
2. 中身をGitHubリポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. GitHub Pagesの反映を待ちます。
4. 先にキャッシュ削除ページを開きます。

https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1

5. その後、下のURLで確認します。

https://cichlid528.github.io/mie-fishing-map/?v=v121-map-label-position-fix

## v121 変更内容

- 「地図ラベル池候補追加」ボタンを、目安位置の候補追加ではなく、表示中の地図ラベル位置を優先して追加する処理に変更しました。
- 岩田池・尺目池など、地図上に見えている「○○池」ラベルの座標を取得できた場合、そのラベル位置へ池候補ポイントを追加します。
- 以前の「地図名池候補リスト」でズレた位置に追加済みの同名候補は、地図ラベル位置が取得できた時に上書き修正します。
- 追加または位置修正後、その池候補ポイントへ地図をズームし、詳細カードを開きます。
- スマートフォンでPBF解析ライブラリが失敗しても動きやすいよう、GeoJSONタイル取得を先に試すようにしました。
- 地図ラベル位置が取得できない時は、ズレた目安位置に勝手に追加せず、「目的の池名が見えるまで拡大して再実行」と案内します。

## 確認方法

1. アプリを開く。
2. タイトル画面の「地図を開く」を押す。
3. 岩田池など、追加したい「○○池」の文字が地図上で見えるまで拡大する。
4. 「地図ラベル池候補追加」を押す。
5. 池候補が追加・更新され、地図がその池候補へズームすることを確認する。
