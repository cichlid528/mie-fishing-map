# v82 edit popup + move spots

## 目的

- 釣果/現地メモの番号を押した時のポップアップ内から、そのまま「この記録を編集」できるようにしました。
- 釣り場詳細カードに「位置調整」ボタンを追加しました。
- 「位置調整」を押した後、地図上の新しい場所をタップすると釣り場ポイントの位置を保存します。
- 固定釣り場は位置補正として保存し、バックアップにも含まれます。
- 自分で追加した釣り場は、その釣り場データ自体の座標を更新します。
- v81までの記録ポップアップ、写真表示、魚種選択、背景透過、追加ポイントは維持しています。
