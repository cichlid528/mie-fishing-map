1. ZIPを解凍します。
2. 中身をGitHubリポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. GitHub Pagesの反映を待ちます。
4. 先にキャッシュ削除ページを開きます。

https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1

5. その後、下のURLで確認します。

https://cichlid528.github.io/mie-fishing-map/?v=v122-center-pond-add

## v122 変更内容

- 「地図中央に池候補追加」ボタンに変更しました。
- 地図上の「○○池」ラベルを画面中央に合わせてボタンを押すと、その中央位置へ池候補ポイントを追加します。
- 国土地理院の池名ラベルを取得できた場合は、仮の池候補名を実際の池名へ自動更新します。
- 池名ラベルを取得できないスマホ環境でも、何も起きない状態を避け、画面中央へ池候補を追加します。
- 追加後は池候補表示に切り替え、追加したポイントへズームして詳細カードを開きます。

## 確認方法

1. マップを開く。
2. 追加したい「○○池」の文字が画面中央付近に来るまで地図を動かす。
3. 「地図中央に池候補追加」を押す。
4. 池候補ポイントが追加され、詳細カードが開くことを確認する。
5. 名前が自動取得できなかった場合は、詳細カードの編集から池名を直してください。
