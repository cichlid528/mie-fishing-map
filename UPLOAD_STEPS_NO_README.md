# v120 アップロード手順

1. ZIPを解凍します。
2. 中身をGitHubリポジトリ `mie-fishing-map` の直下へ上書きアップロードします。
3. GitHub Pagesの反映を待ちます。
4. 先にキャッシュ削除ページを開きます。

https://cichlid528.github.io/mie-fishing-map/reset-cache.html?auto=1

5. その後、下のURLで確認します。

https://cichlid528.github.io/mie-fishing-map/?v=v120-map-label-pond-candidates

## 修正内容

- スマートフォンで「地図名池候補追加」ボタンを押した時に、岩田池・尺目池など名前付きの池候補を即時追加できるようにしました。
- click / touchend / pointerup をまとめて受けるようにして、スマホのタップでも確実に処理を開始します。
- 地図上の表示範囲から、国土地理院注記の「池」を含む名前を池候補として追加します。
- 取得範囲が広すぎる時は、スマホで止まらないよう地図中央周辺を優先して取得します。
- 通信や端末の制限で地理院注記を取得できない時も、表示範囲に近い補助池候補を追加して、何も反映されない状態を避けます。
- 追加後は自動で「釣り場」タブと「池候補」表示に切り替え、スマホではメニューも開きます。


## v120 変更内容

- 「地図名池候補追加」ボタンで、外部タイル解析に頼らず名前付き池候補リストから即時追加します。
- 津市周辺の例として、岩田池・尺目池を候補に含めました。
- 追加後は自動で「池候補」表示へ切り替えます。
- 位置は候補用の目安なので、詳細カードの位置調整で直せます。
