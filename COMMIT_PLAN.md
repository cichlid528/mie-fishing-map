# Commit Plan

## まとめてコミットする場合

```text
feat: 釣果記録に野池バス分析項目を追加
```

## 分けてコミットする場合

```text
feat: 釣果記録に水位・ベイト・カバー項目を追加
feat: ルアー詳細と実寸サイズ記録を追加
feat: 記録一覧に40up集計を追加
chore: PWAキャッシュをv44に更新
```

## GitHubに入れる前の確認

- まず新しいブランチで確認
- 公開中の `index.html`、`app.js`、`style.css` を上書きする前にバックアップ
- スマホでホーム画面追加済みの場合、更新後にアプリを再起動
- 大事な釣果データはアプリ内の「バックアップ保存」でJSONを書き出してから作業
